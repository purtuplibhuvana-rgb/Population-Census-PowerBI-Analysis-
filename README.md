# Population Census Power BI Analysis

Interactive demographic analysis dashboard built using Census 2011 data with Power BI.

## Project Overview
This project analyzes demographic patterns such as population distribution, gender ratio, literacy rate, and workforce participation across Indian states using an interactive Power BI dashboard.

## Tools Used
- Power BI
- Microsoft Excel
- Data Visualization

## Key Insights
- Workforce participation rate: **47.44%**
- Literacy rate: **~59%**
- Male population slightly exceeds female population across most states
- Significant regional differences in employment and literacy levels
